# Python Strings Practice

This project is for practicing Python string concepts using simple examples.

## What the program does
- Takes a student's full name as input
- Removes extra spaces
- Counts the number of letters (excluding spaces)
- Creates a simple student ID using initials and letter count

## What I learned
- Python string methods
- Indexing and slicing
- Using input() and print()
- Basic problem-solving with strings
